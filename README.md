# Python Learning Journey 🐍

This repository documents my journey of learning Python from scratch.

The goal of this project is to track my progress, experiment with new concepts, and build a solid foundation in Python programming.

## Repository Structure

### `day1`

Contains the exercises, notes, and code from my very first day learning Python.

### `basics_learning_process`

A collection of experiments, practice snippets, and commented examples created while learning Python fundamentals.

These files serve as a record of my learning process and help me revisit concepts as I progress.

### Other Folders

Each folder focuses on a specific topic, exercise, or project and includes its own README with instructions on how to run the code.

## Topics Covered

This repository will gradually expand to include:

- Variables and data types

- Conditional statements
- Loops
- Functions
- Lists, tuples, and dictionaries

## Running the Code

Navigate to the folder you want to explore and follow the instructions in its README.

Example:

```bash
cd folder_name
python main.py
```
